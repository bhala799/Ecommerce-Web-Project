# 🛒 E-Commerce Web Application

A responsive E-Commerce Web Application developed using HTML, CSS, and JavaScript.  
This project demonstrates frontend development skills including authentication, API integration, and cart management using LocalStorage.

---

## 🚀 Features

- User Registration with Email & Password Validation
- User Login Authentication
- Session Handling using LocalStorage
- Product Listing using DummyJSON API
- Search Functionality (by title & category)
- Product Details Page
- Add to Cart & Remove from Cart
- Dynamic Total Price Calculation
- Customer Reviews Section

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Bootstrap Icons  
- DummyJSON API  
- LocalStorage  

---

## 🔐 Authentication System

- Registration with Regex validation
- Login credential verification
- Protected pages (Login required before accessing Home/Cart)
- Session stored using LocalStorage

---

## 🛒 Cart System

- Add products to cart
- Remove products from cart
- Total price calculation
- Cart data stored in LocalStorage

---

## 🌐 API Used

Product data is fetched from:  
https://dummyjson.com/products

---

## ▶ How to Run

1. Clone the repository
2. Open `index.html` in your browser
3. Register a new account
4. Login and start shopping
